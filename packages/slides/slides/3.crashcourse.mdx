## What have we learned?

<ul>

<Appear>

<li><Code>docker</Code> is the main command used.</li>
<li>To execute something, we need to use the <Code>docker run</Code> command</li>
<li>To build something, we need to use the <Code>docker build</Code> command</li>
<li>To see what docker processes (containers) are running use the <Code>docker ps</Code> command</li>
<li>To kill a process, we need to use the <Code>docker kill</Code> command</li>
<li>To execute code on an already existing container, we need to use the <Code>docker exec</Code> command</li>
<li>All of these commands have a lot of arguments</li>

</Appear>

</ul>
