<Layout header>

# $ docker exec

</Layout>

---

<Layout>

## Arguments: 

<ul>

<Appear>

<li><Code>&lt;id&gt;</Code>: The ID or name of the container</li>
<li><Code>&lt;command&gt;</Code>: The command to be run</li>

</Appear>

</ul>

</Layout>

<Layout header>

# $ docker ps

</Layout>

---

<Layout>

## Arguments: 

<ul>

<Appear>

<li><Code>-a</Code>: Will list all containers (including inactive)</li>
<li><Code>-q</Code>: Will only print the IDs of the containers</li>

</Appear>

</ul>

</Layout>

--- 

<Layout header>

# $ docker kill

</Layout>

---

<Layout>

## Arguments: 

<ul>

<Appear>

<li><Code>&lt;id(s)&gt;</Code>: The ID (or list of IDs) of containers to be killed</li>

</Appear>

</ul>

</Layout>
