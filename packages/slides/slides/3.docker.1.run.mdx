## Arguments: 

<ul>

<Appear>

<li><Code>--name</Code>: Will give the container a name</li>
<li><Code>-i</Code>: Will run in interactive mode</li>
<li><Code>-t</Code>: Will run in tty mode (will allow terminal use while interactive)</li>
<li><Code>-it</Code>: A combination of the above. In translation: open a terminal connection</li>
<li><Code>-d</Code>: Will run in daemon mode (background process)</li>
<li><Code>--rm</Code>: Will remove the container on exit</li>
<li><Code>&lt;command&gt;</Code>: Will override the default command</li>

</Appear>

</ul>
