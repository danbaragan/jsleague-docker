## Arguments: 

<ul>

<Appear>

<li><Code>-t</Code>: Will give the image a name</li>
<li><Code>--build-arg</Code>: An argument used in build steps</li>
<li><Code>-f</Code>: The name of the Dockerfile (ex: Dockerfile.prod)</li>
<li><Code>--no-cache</Code>: Attempt to build with no cache</li>
<li><Code>&lt;path&gt;</Code>: The path to the Dockerfile / context</li>

</Appear>
